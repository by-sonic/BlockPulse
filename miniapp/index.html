How It Works

  1. Detect — we identify your region and ISP via GeoIP
  2. Probe — your browser attempts TLS handshakes to VPN ports using fetch() timing analysis
  3. Analyze — a fast error (~<300ms) means the port is blocked; a slow error (>300ms) means TLS handshake happened (port reachable); a timeout means DPI filtering
  4. Aggregate — results are sent to the server and combined with other users' data to build a real-time reachability map

Privacy

Your IP address is used only for GeoIP lookup (region/city/ISP). No personal data is collected or stored. All probe results are anonymized and aggregated.
